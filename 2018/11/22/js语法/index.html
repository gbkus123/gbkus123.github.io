1.语句 和 表达式

2018年11月22日20:17:16
语句是为了完成某种任务的操作,一般情况下一条语句不需要返回值,比如下面就是一条赋值语句

var a;

语句以;为结尾,而;之前可以没有任何内容,javascript 将其视为空语句,可以这么写,但不推荐:

;;;

表达式是一条期望得到值得语句,一条语句可以有多条表达式,这是一条有表达式的语句:

var a = 123-122;

而表达式本身 不需要;结尾,因为如果添加了分号, javascript 引擎会把它视为语句,但他仅有返回值,没有任何操作 没有任何意义

//一下语句虽然经过计算,但无意义
1+2;
'abc'+'d';

2.变量

变量是对值得引用,使用变量等于引用一个值,每一个变量都需要一个变量名;

声明变量用var

声明和赋值是两个步骤:

var a = 1;
//其实是首先先创建一个变量
var a;
//然后在变量和a 和数值1之间建立引用关系
a = 1;

如果声明一个变量没有赋值那么,这个变量的值是不存在的, js 用undefined 表示

//在浏览器的console 中如果,输入 var a;
var a;
//console 中会直接返回undefined

可以一次声明多个变量,用,分隔

var a ,b;
var a =1 ,b=2;

javascript 和 Php 一样都是动态语言, 变量没有类型限制:

var a =1;
a = {1:1};
a='a';

重新声明一个变量是无效的(不会报错),但重新声明时可以赋值并覆盖之前变量的值,不推荐这么写

var  a = 1;
var a; //无效
var a = 2;//a等于2 不推荐 , 正常写法: a = 2;

3.变量提升

javascript 引擎的工作方式,先解析代码,并获取所有声明变量的语句,相当于把所有变量的声明都提升到代码的头部,这就是变量提升,变量提升只对var声明的有效, 如果声明一个新变量不用var,是会报错的,比如:new = 1;这种写法不适用于声明变量,但可以作为对象的属性赋值

console.log(gg);
var gg = 1; 
//显示 undefined,这是因为实际运行的代码是:
//var gg;
//console.log(gg);
//gg = 1;

1.4标识符

标识符标识符就是个名字,用来识别不同对象,可以理解为变量名和函数名.js 的标识符大小写敏感, a 和 A ,a()和A()都是不同的变量和函数

标识符有明确的命名规则,不符合规则js 解释引擎就会报错:

第一个字符，可以是任意Unicode字母（包括英文字母和其他语言的字母），以及美元符号（$）和下划线（_）。

第二个字符及后面的字符，除了Unicode字母、美元符号和下划线，还可以用数字0-9。

var _a = 1;
var $_1 =1;
var _1 = 1;
var $1 = 1;
var a1 = 1;
var $a = 1;
var _a = 1;
var a_$1 = 1;
var π =1 ;    
var 中文 = 1;
var ##qwe =1; //报错
var 1 = 1;//报错
var a-1 = 1;//报错
